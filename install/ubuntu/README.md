# Ubuntu Installation Scripts (Stub Implementation)

このディレクトリは、Ubuntu環境用のインストールスクリプトとテストを配置する場所です。

## 現在の状態

**仮実装（Stub）**: 現時点では、Ubuntu環境での実用優先度が低いため、スクリプトは実装されていません。

## 今後の実装予定

Ubuntu環境でのセットアップが必要になった際に、以下のスクリプトを実装します：

- パッケージ管理（apt）
- 開発ツールのインストール
- 環境固有の設定

## 注意事項

- CI/CDワークフローでは、このディレクトリは現在除外されています
- OS共通のスクリプト（chezmoi等）は `install/common/` に配置されています
